Simple web app that allows intern doctors working at University Hospital Limerick to check if they are scheduled to carry the cardiac arrest bleep for an upcoming on-call shift, based on date and shift-type.


## DISCLAIMER

This app is intended for use as a quick reference-tool only. 

All query results are based on the revised "Intern On Call Rota Structure" document which was circulated by UHL's Medical Manpower department, via email, on **27/02/2026**. 

This is a static website, which means the Ward Distributions and Scheduled Arrest Bleep days were entered into the manually. If the rota structure or arrest bleep assignments undergo further revisions in the future, then results provided by this app will be inaccurate.
