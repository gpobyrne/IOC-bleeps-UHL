Simple web app that allows intern doctors working at University Hospital Limerick to check if they are scheduled to carry the cardiac arrest bleep for an upcoming on-call shift, based on date and shift-type.


## DISCLAIMER

This app is intended for use as a quick reference-tool only. 

Currently, all query results are based on the revised Intern On Call Rota Structure document that was distributed by UHL's Medical Manpower department (via email) on 27/02/2026. 

This is a static webpage, meaning that it must be manually updated. 

  If the Ward Distributions or Arrest Bleep Schedule change in the future, then the results of this page will be inaccurate.
